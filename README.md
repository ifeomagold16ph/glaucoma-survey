# Public Health Data Analysis Repository

This repository contains Python-based data analysis projects in public health, including life expectancy and eye health.

## Projects

### 1. Life Expectancy Analysis (2000–2015)
- Explored WHO global life expectancy data.
- Objectives:
  - Analyze trends over time
  - Examine health inequalities
  - Investigate GDP & healthcare expenditure impacts
- Tools: Python, Pandas, NumPy, Matplotlib
- Key Steps: Data cleaning, EDA, Visualization
- Summary: Insights into global health patterns and socioeconomic factors.

**Notebook:** `Life_Expectancy_Analysis.ipynb`

### 2. Global Trends in Vision Impairment
- Analyzed 10,000 patient records to identify trends in eye disease.
- Key insights:
  - Cataract as leading predictor for surgery
  - Associations between refractive error and spectacle prescription
- Tools: Python, pandas, matplotlib

**Notebook:** `Global_Trends_in_Vision_Impairment_A_Data_Analysis.ipynb`

### 3. Glaucoma Survey Data Analysis
- Exploratory data analysis on glaucoma survey responses using Python.
- Tools: Python, pandas, matplotlib

**Notebook:** `Glaucoma_survey.ipynb`
